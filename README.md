# Casper Game Filter

## Быстрый старт
1. Запусти программу (CasperGameFilter).
2. Настрой фильтры ползунками.
3. Нажми **Сохранить шаблон** и при желании назначь горячую клавишу.
4. Быстрые пресеты: **Стандарт / Светлее / Темнее**.

## Горячие клавиши
- В настройках можно задать клавишу для открытия окна из трея.
- Для пользовательских шаблонов можно назначать свои клавиши.
