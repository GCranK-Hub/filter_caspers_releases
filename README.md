# Casper Game Filter

## 🚀 Быстрый старт
1. Запусти CasperGameFilter
2. Настрой фильтры с помощью ползунков
3. Нажми «Сохранить шаблон» и при необходимости назначь горячую клавишу
4. Быстрые пресеты: **Стандарт / Светлее / Темнее**.

## ⌨️ Горячие клавиши
- В настройках можно задать клавишу для открытия окна из трея
- Для пользовательских шаблонов можно назначать отдельные горячие клавиши

----------------------------------------------------------------
## 🔄 Автообновление
- Программа автоматически проверяет обновления при запуске
- Ручную проверку можно выполнить в настройках
